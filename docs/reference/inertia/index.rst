=====================
InertiaJS Integration
=====================

InertiaJS is a framework for building modern web applications. It allows you to build a single-page
application with a traditional server-side routing setup, but with the benefits of a modern
JavaScript framework.

This API reference is automatically generated from the docstrings in the code, and is useful for
finding out what methods and attributes are available for a given class. The API reference is
divided into several sections, each of which is listed below.

.. note:: Private methods and attributes are not included in the API reference.
